# Project GivingCare CareGuide Website

A responsive, digital-first caregiver resource hub for Singapore.

## Files
- `index.html` — website content
- `styles.css` — design system and responsive layout
- `script.js` — search, filters, mobile menu and printable toolkit sheets

## Preview
Open `index.html` in any modern browser.

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all three website files to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/root`, then save.

## Before public launch
- Replace the placeholder CareFest programme with confirmed date, venue, speakers and schedule.
- Confirm the official Project GivingCare email and social links.
- Add the approved Project GivingCare logo if available.
- Ask clinical, legal and partner organisations to review relevant pages.
- Recheck all external links and eligibility criteria.
- Add an accessibility and privacy statement if analytics or forms are introduced.
