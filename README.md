# Project GivingCare HTML Website

This version uses only HTML, CSS and JavaScript.

## Publish on GitHub Pages
Upload the contents of this folder directly to the repository root.

The repository should contain:
- index.html
- about.html
- careguide.html
- carefest.html
- resources.html
- partners.html
- contact.html
- styles.css
- script.js
- assets/
- guides/

Do not upload the enclosing `project-givingcare-html` folder itself.

Then go to:
Settings → Pages → Deploy from a branch → main → /root

No build step, Node.js or Next.js is required.
