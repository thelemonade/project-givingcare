# Project GivingCare Website

A modern multi-page website built with Next.js.

## Included pages
- Home
- About
- CareGuide library
- Six in-depth caregiver guides
- CareFest 2026
- Resources
- Partners
- Contact

## Run locally
1. Install Node.js 18 or later.
2. Open this folder in Terminal.
3. Run:
   npm install
   npm run dev
4. Open http://localhost:3000

## Build a static export
Run:
   npm run build

The exported static website will be created in the `out` folder.

## Publish
You can deploy the project through Vercel, Netlify or GitHub Pages after building the static export.

## Content review before launch
All medical, legal, financial and service information should be reviewed by relevant professionals and partner organisations. Recheck official links and eligibility criteria shortly before CareFest.
